Project 3 ‚Ä¢ Concentration

Concentration is a puzzle-based game where the user makes matches on the board to reveal a rebus puzzle. The user will be given a prompt on each card match that includes an opportunity to guess the riddle hidden behind the cards.


Built with Javascript + HTML + CSS

Concentration's gameplay comes from the TV Game Show "Classic Concentration". The game is based on the children's card game called Memory. It aired on and off from 1958 to 1991.

Built by Julie

Concentration was my first project ever built. It is a solo project that I created in a seven-day sprint. The project aimed to display basic concepts of Javascript, such as event listeners, functions, loops, and DOM manipulation.


Project Features.

    This project features heavy CSS and set time-outs to simulate animations on card flips.

    I created each unique puzzle using Photoshop.

    Modal popups prompt the user to guess the puzzle after a match is made. Once the riddle behind the cards is correctly guessed, another modal generates, revealing the puzzle image and your game stats.



What I Learned

In creating Concentration, I ran into the problem of figuring out how to simulate animation with only knowing basic Javascript, HTML, and CSS. I learned how to utilize timeouts and CSS tricks to create an animation.
